# Proyectos Personales con React.js 🚀

¡Bienvenido/a a mi repositorio de proyectos personales en React.js! Aquí encontrarás una colección de aplicaciones y sitios web que he creado para mejorar mis habilidades de desarrollo frontend utilizando React.

## Proyectos Destacados

### [Proyecto 30: Tienda de Comida](https://tienda-y-paneldeadministracion-dru.vercel.app/)
Tienda de comidad donde poder realizar pedido en tiempo real, con panel de adminsitración ("/administracion") para gestionar los pedidos que se almacenan en una base de datos. Front-end craedo con React.js y Tailwind CSS. Back-end y DB creado con Prisma y MongoDB.

### [Proyecto 24: GuitarLA]
E-commerce de guitarras con todas las funcionalidades. Front-end creado 3 veces con diferentes frameworks (Astro, Remix y Next.js). Back-end y DB creado con Strapi y PostgresSQL.

## ¿Cómo explorar?

Cada proyecto tiene su propio directorio en este repositorio. Siéntete libre de explorar los proyectos que más te interesen.

## Tecnologías más Utilizadas

- HTML5, CSS3, JavaScript, Tailwind CSS, React.js, Next.js

## ¿Cómo ejecutar los proyectos localmente?

1. Clona este repositorio: `git clone https://github.com/tu-usuario/react-projects.git`
2. Navega al directorio del proyecto que te interese: `cd nombre-del-proyecto`
3. Instala las dependencias: `npm install`
4. Inicia la aplicación: `npm start`
